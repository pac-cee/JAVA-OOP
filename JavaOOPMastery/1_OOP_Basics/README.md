# OOP Basics in Java

## What are Classes and Objects?
- **Class**: Blueprint for objects. Example: `Car`, `Student`.
- **Object**: Instance of a class. Example: `myCar`, `johnStudent`.

## Example
```java
public class Car {
    String brand;
    int year;
    void drive() {
        System.out.println(brand + " is driving!");
    }
}
```

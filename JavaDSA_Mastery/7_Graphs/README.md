# Graphs in Java

## What is a Graph?
- Set of nodes (vertices) connected by edges.
- Types: Directed, Undirected, Weighted, Unweighted

## Applications
- Social networks, maps, web page links

## Example: Adjacency List
```java
import java.util.*;
List<List<Integer>> adj = new ArrayList<>();
```

# Trees in Java

## What is a Tree?
- Hierarchical data structure with root, nodes, and leaves.
- Types: Binary Tree, Binary Search Tree (BST), AVL Tree, etc.

## Example: Binary Tree Node
```java
class TreeNode {
    int data;
    TreeNode left, right;
    TreeNode(int data) { this.data = data; }
}
```
